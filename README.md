# Max3D

CoreXY 320 × 320 × 300 mm 3D printer. BTT SKR Mini E3 V3.0 (STM32G0B1), TMC2209, 12 V.

## Repository layout

| Path | Contents |
|------|----------|
| `Klipper/` | Klipper `printer.cfg` (Orange Pi Zero2W host, Mainsail UI) |
| `Marlin/` | Marlin `Configuration.h` / `Configuration_adv.h` (alternative firmware) |
| `ESP3D/` | ESP3D config for ESP32-C3 WiFi bridge + setup guide |
| `docs/` | Enclosure wiring pinout, board schematics and datasheets |
